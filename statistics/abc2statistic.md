# basic #
for *n* items set
$$ { x_1, x_2, x_3, ... x_n } $$    

mean:
$$ \mu=x_{mean} = \frac {\sum_i^n x_i} {n}  $$

variance:
$$    \sigma^2=x_{variance} = \frac {\sum_i^n (x_i-x_{mean})^2}{n}  $$

standard variance:
$$ \sigma = \sqrt{\sigma^2} = \sqrt{\frac {\sum_i^n (x_i-x_{mean})^2}{n} } $$
